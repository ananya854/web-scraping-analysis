# Web Scraping & Company Analysis

An end-to-end data pipeline that scrapes, cleans, and visualizes US company data.

## What it does
- Scrapes company data from the web using Python
- Cleans and structures the dataset with Pandas
- Outputs an interactive HTML dashboard for exploration

## Tech Stack
`Python` `BeautifulSoup / requests` `Pandas` `Jupyter Notebook` `HTML` `JavaScript`

## Files
- `Web_Scraping.ipynb` — Full scraping + analysis notebook
- `us_companies_analyzer.html` — Interactive visualization dashboard

## How to run
```bash
pip install requests beautifulsoup4 pandas
jupyter notebook Web_Scraping.ipynb
```
